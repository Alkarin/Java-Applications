Take away game project

"Takeaway" is a two-player game. The game consists of one or more piles of coins. A player may remove any number of coins from a single pile when it is their turn. The player to remove the last coin wins.

Implementation

Running the program

The program is invoked from the command-line as: 

     java cisc187.takeaway.{@link cisc187.takeaway.PlayController} [ai] [random]
Where:
PlayController
is the name of the main class where execution begins
ai
is an optional argument that indicates player #2 is the computer.
random
an optional argument than creates a random number of piles and random coins per pile. For non-random games, the user is prompted for these values.
