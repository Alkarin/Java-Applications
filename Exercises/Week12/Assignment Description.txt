Snakes and Ladders, part 1.

See https://en.wikipedia.org/wiki/Snakes_and_Ladders

Subject to change

Create an abstract class called BoardElement, which has abstract methods nextElement and print.

Create a class called SnakeElement from BoardElement where moveTo returns how many location to move backwards and print returns a "\".

Create a class called LadderElement from BoardElement where moveTo returns how many location to move backwards and print returns a "#".

Create a class called RegularElement from BoardElement where moveTo returns zero and print returns a ".".


Create a class called Board which has a 10 by 10 array to hold BoardElements. Then fill the Board with RegularElements; and for each row one LadderElement or one SnakeElement. After that, create a main method that prints out the board using the print methods on the elements.


Submit one .java file and one .txt file of the output.
